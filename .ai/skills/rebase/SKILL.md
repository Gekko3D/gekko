# Skill: Rebase Documentation Deltas

Use this skill after a task has landed and its temporary notes should become durable docs.

## Steps

1. Read the feature context and any linked status report.
2. Apply durable doc changes into the correct canonical file under `docs/` or `.ai/docs/`.
3. Preserve existing structure and avoid unrelated rewrites.
4. Mark the feature context as rebased or closed once the durable docs are current.
