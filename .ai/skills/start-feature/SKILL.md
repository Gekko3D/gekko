# Skill: Start Feature Or Task

Use this skill at the start of non-trivial work.

## Steps

1. Read `.ai/context.md`, `.ai/conventions.md`, `AGENTS.md`, and the smallest matching docs under `docs/`.
2. Normalize the task: objective, scope, owner bucket, constraints, expected artifacts.
3. State the invariant that must remain true.
4. Choose the smallest verification command before editing.
5. Create or update `.ai/features/[task].md` from `_FEATURE_TEMPLATE.md` when the task is non-trivial.
6. If confidence is not `High`, write a status report under `reports/` before broad edits.
