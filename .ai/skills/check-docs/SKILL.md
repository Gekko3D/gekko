# Skill: Check Documentation Tracking

Use this skill when a task may have changed docs or workflow expectations.

## Steps

1. Compare the active feature context, status report, and current diff.
2. Check whether the change should update `docs/`, `AGENTS.md`, `.ai/docs/`, or `CHANGELOG.md`.
3. Flag missing updates explicitly.
4. Prefer updating canonical docs in the same task when the behavior has already changed.
