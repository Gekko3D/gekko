# Skill: PR Review

Use this skill for findings-first review.

## Steps

1. Read `.ai/context.md`, `.ai/conventions.md`, and `AGENTS.md`.
2. Review the diff against the relevant runtime, renderer, content, or workflow invariants.
3. Prioritize findings on correctness, regressions, missing tests, risky assumptions, and doc drift.
4. Report findings first with concrete file references and verification gaps.
5. Mention residual risks if the diff looks correct but was not fully validated.
