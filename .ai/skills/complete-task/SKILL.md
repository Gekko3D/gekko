# Skill: Complete Task

Use this skill when wrapping up work.

## Steps

1. Update the active feature context with what changed, invariant preserved, and verification run.
2. Update canonical docs if behavior or workflow expectations changed.
3. Add a changelog entry when the repo-level workflow or public engine behavior changed materially.
4. Call out anything intentionally not verified.
5. Do not push, create a PR, or rewrite unrelated files unless explicitly asked.
